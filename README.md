## Antes de ejecutar es necesario instalar dependencias ocupadas:

```bash
npm install lucide-react
npm install react-sketch-canvas
```


## Para ejecutar el server:

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en el navegador.
